Controls

  • WASD / Arrow Keys: Move camera
  • Mouse Wheel: Zoom in/out
  • Mouse Click: Paint selected tile
  • Hover: Highlight tile

About

A tile map sandbox for visualizing and placing hex terrain. Built with Rust + Bevy.
